# Example: Conflict Resolution

## User request

请优先根据讲义解释这个概念，但如果讲义与当前项目实现不一致，请明确指出冲突，不要强行揉成一个答案。

## Expected shape

- identifies the preferred source
- checks current project/runtime evidence
- surfaces the conflict explicitly
- separates source-derived explanation from project-reality correction
- avoids a falsely unified answer

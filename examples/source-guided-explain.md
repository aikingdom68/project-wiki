# Example: Source-Guided Explanation

## User request

我在做一个讲解题目的系统，我有一个例题库。请用 project-wiki 讲解这道题时优先调用例题库里的知识，并尽量按例题的思路来讲。

## Expected shape

- identify example bank as primary source
- explain in the example bank's reasoning order when possible
- separate source-derived explanation from local supplemental and general supplemental explanation
- do not jump straight to a generic answer
